Events in JS

1. Mouse events : These events occur when a user interacts with the mouse.

            Other Mouse Events
            a.dblclick :Double-clicking an element.
            b.mousedown: When the mouse button is pressed down.
            c.mouseup : When the mouse button is released.
            d.mouseover : When the mouse enters an element.
            e.mouseout: When the mouse leaves an element.

2. Keyboard events : Triggered when the user presses or releases a key.

            Other Keyboard Events
            a.keydown: When a key is pressed down.
            b.keyup: When a key is released.

3. Form events : These events are triggered when interacting with form elements.

            input : when user interact with inputField
            
            Other Form Events
            a.submit:When a form is submitted.
            b.focus:When an input field gains focus.
            c.blur:When an input field loses focus.
            d.change: when user modified an inputField

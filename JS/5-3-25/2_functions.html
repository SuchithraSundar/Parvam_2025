3 Ways to declare a function

1. function declaration 

Syntax:
                function function_name(parameter){
                    //code
                }
            
                call a function:
                function_name(argument);

2. function expression 

Syntax:
                
                const  function_name = function(parameter){
                    //code
                }
            
                call a function:
                function_name(argument);

3. arrow function 

Syntax:
                
                const  function_name = (parameter) => {
                    //code
                }
            
                call a function:
                function_name(argument);

4 types of function

1. function accepting a parameter and not returning value

2. function accepting a parameter and returning value

3. function not accepting a parameter and not returning value

4. function not accepting a parameter and returning value

Array Methods

1. foreach - Iterates over each element in an array.

2. map - Creates a new array by applying a function to each element.

3. filter - Returns a new array containing only elements that pass a condition

4. reduce - Reduces an array to a single value.

5. sort - Sorts elements in place

6. splice(start,delete,insert) - Adds/removes elements at a specific index.

7. every -

8. some - Checks if at least one element satisfies a condition.

M - model(tables)
V - views(templates)
C - controller(logic)


DTO -Data transfer object


Dependencies:
1. Spring Web(web) - Build web, including RESTful, applications using Spring MVC. Uses Apache Tomcat as the default embedded container.
2. MySQL DriveR(SQL) - MySQL JDBC driver.
3. Spring Data JPA (SQL)- Persist data in SQL stores with Java Persistence API using Spring Data and Hibernate.
4. Thymeleaf(Template Engine) -A modern server-side Java template engine for both web and standalone environments. Allows HTML to be correctly displayed in browsers and as static prototypes
5. Lombok(Developer Tools) - Java annotation library which helps to reduce boilerplate code.
